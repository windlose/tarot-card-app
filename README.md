# 🔮 群星的愚言 - 塔罗牌占卜

一个沉浸式的塔罗牌占卜H5应用，支持在微信中分享。

## ✨ 功能特点

- 🎴 **22张大阿卡纳牌** - 完整的塔罗牌组
- 🎬 **视频动态背景** - 精美的星空动画背景
- ✨ **流畅的抽牌动画** - 点击卡牌后移动到中心再翻牌
- 📖 **正逆位解读** - 每张牌都有正位和逆位的含义
- 📱 **移动端适配** - 可在手机浏览器和微信中使用

## 🎮 使用方法

1. 打开页面
2. 从底部牌阵中选择一张牌
3. 观看抽牌动画
4. 查看牌义解读
5. 点击"再次占卜"可重新抽牌

## 🛠️ 技术栈

- HTML5
- CSS3 (动画、Flexbox布局)
- 原生JavaScript
- 视频背景
